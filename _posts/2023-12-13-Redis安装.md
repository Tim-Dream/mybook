* 安装
``` shell
yum install redis -y
```

* 启动
``` shell
systemctl start redis
```
* 重启
``` shell
systemctl restart redis
```
## 修改默认配置

* 进入文件修改
``` shell
vim /etc/redis.conf
```

*  开启远程链接
``` shell
#bind 127.0.0.1
protected-mode no
daemonize yes
```
* 设置密码和端口
``` shell
requirepass Jj20180901!
port 6381
```
